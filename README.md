## Simple Java Programs

This project contains 5 simple Java programs for learning and practice:

1. **SmallScriptDay2.AgeInDays.java** - Converts age from years to total days
2. **SmallScriptDay2.BMICalculator.java** - Calculates Body Mass Index (BMI) from weight and height
3. **SmallScriptDay2.SimpleInterest.java** - Calculates simple interest on a principal amount
4. **SmallScriptDay2.TemperatureConvertor.java** - Converts temperature between Celsius and Fahrenheit
5. **SmallScriptDay2.TipSplitter.java** - Splits a bill and calculates tip amounts

All programs are located in the `src/` directory and can be run through the Main.java file or individually.
