## Simple Java Programs

This project contains 5 simple Java programs for learning and practice:

1. **AgeInDays.java** - Converts age from years to total days
2. **BMICalculator.java** - Calculates Body Mass Index (BMI) from weight and height
3. **SimpleInterest.java** - Calculates simple interest on a principal amount
4. **TemperatureConvertor.java** - Converts temperature between Celsius and Fahrenheit
5. **TipSplitter.java** - Splits a bill and calculates tip amounts

All programs are located in the `src/` directory and can be run through the Main.java file or individually.
